The @keyframes rule in CSS


Your average block of CSS looks something like this:

        
            .class{
                background-color:red;
                color:orange;
                position:relative;
                border:solid 1px;
            }

But what if you want the ".class" class to move? Well, you can use the @keyframes rule to move it.



What even are keyframes?

Keyframes is a rule in CSS used to move things around. For example, copy-and-paste this code into your CSS file and see how it works: 


        
            .class{
                animation:keyframe 3s infinite;
            }
            @keyframes keyframe{
                0%{font-size:50px;}
                100%{font-size:100px;}
            }
