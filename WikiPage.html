BlokRace is a website based on squares on the screen "racing". It was made by the Hedron company in late 2023. Its original purpose was to be a test for keyframe animation, but we saw more potential in this, so the test was now a website.

The layout

The original layout consisted of the races being displayed on the same page, but it was really bad. unfortunately, we don't have any images of that. We made sure there was almost nothing happening on the screen. We were delighted with the final product. The user interface looked really nice, and at the time of this wiki page's launch, we will have finished the software.
